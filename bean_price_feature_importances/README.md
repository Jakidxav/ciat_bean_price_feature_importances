This is the repository that holds my Jupyter Notebooks containing my analysis of bean price feature importances within the PABRA program.

Information for the data used in this study can be found at the following links:- [WorldClim](https://www.worldclim.org/data/monthlywth.html) global monthly temperature and precipiation data
- [FAO GIEWS](https://fpma.apps.fao.org/giews/food-prices/tool/public/#/dataset/domestic) domestic food prices for beans
